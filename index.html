Clinic Website with Appointment Booking System

This project is for my final submission to Harvard's CS50 Web Programming with Python and Javascript

This a Django-based web application created for a pseudo-clinic called Radiant, designed to manage the clinic's appointments efficiently. It provides features for booking appointments, managing appointment statuses, and sending confirmation emails to patients. And being a website, it also contain all of the clinic's services, offers, staff, etc,. Informations and images contained in this project are AI generated just to populate the pages.

POS Software for a Small Restaurant

This is a simple POS system I built for my brother's small restaurant.

Built with CustomTkinter GUI, it has dark and light theme. Clicking a Menu category shows the menu items accordingly. The buttons rely on the database written in JSON, the user must input the menu and menu items accurately in .json format, the buttons then are updated accordingly. This is my first personal python program, I built it for my CS50p's final project.

Youtube Playlist Downloader

A simple but powerful downloader for Youtube. It allows the user to download all the contents of a youtube playlist through the playlist's link

With one click, the user can get all the videos he want, either in video or audio file type. Single file download is also available. The user can choose a prefered folder for the download location. It also has a cancel button to cancel the ongoing download

Ebay-like Ecommerce Auction App

This Ebay-like app is a commerce auction site built with Django and Bootstrap for the components.

The app allows the users create an account, post listings, place bids on listing, comment on those listings, and add listings to a 'watchlist'. The admin/superuser can utilize Django's admin page to manipulate the data.
